What you will learn in courses that focus on student improvement
ما الذي ستتعلمه في الدورات التدريبية التي تركز على تحسين مستوي الطالب

  • 1

    تعلم لغة HTML

    لبدأ حياتك في مجال ال Web يجب عليك تعلم لغة تصميم الصفحات والتي لا يمكن الإستغناء عنها سواء كنت Designer أو Developer وهي لغة Html وسوف تجد الدورة التعليمية ومعها جميع المراجع التي تحتاجها ومعها تمارين وأفكار كثيرة

  • 2

    تعلم لغة CSS

    لغة CSS هي اللغة المسؤولة عن تنسيق صفحات ال Web وهل التي تتيح لك إستعمال الألوان والخطوط والأبعاد وكل ما يجعل من الصفحة الشكل الجميل الذي تراه ولا غنى عنها طبعا وسوف تجد الدورة التعليمية ومعها جميع المراجع التي تحتاجها ومعها تمارين وأفكار كثيرة

  • 3

    تعلم لغة JavaScript

    تعلم اساسيات لغة JavaScript وما يخصها مع المراجع

  • 4

    تعلم مكتبة jQuery

    واحدة من أسهل وأجمل المكتبات التي تتعامل مع صفحة ال Web والتي تسهل عليك الكثير من الأمور التي يتم عملها بصعوبة بلغة ال JavaScript فقط وشعارها Write Less Do More وعن طريقها يمكنك أن تتحكم في العناصر وتقوم بعمل العمليات الحسابية التي تفيدك في التصميم

  • 5

    تعلم لغة PHP مع ال OOP والمراجع

    لغة PHP من أشهر اللغات لبناء تطبيقات ومواقع الويب وهي لغة من لغات ال Scripting Language ولها دعم كبير وهي مجانية وسهلة التعلم جدا والكثير من المواقع العالمية مبرمجة بهذه اللغة والدورة التعليمية هنا سوف تفيدك جدا في اتقان اللغة باذن الله.

  • 6

    تعلم Command Line مع برنامج Cmder

    تعلمك لل Command Line سوف يجعلك تتطور كثيرا في عملك وخصوصا في مجال ال Development والمجالات الأخرى مثل ال System Administration وسوف توفر عليك الكثير من الوقت والجهد عن طريق الأشياء التي يمكن عملها بواسطة ال Commands
